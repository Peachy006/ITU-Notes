Int - heltal
double - kommatal
char - en karakter
string - tekst
boolean - sandt/falsk   - ja/nej
- `byte` → meget små heltal (fra -128 til 127).
    
- `short` → små heltal (fra -32.768 til 32.767).
    
- `int` → “normale” heltal (fra ca. -2 mia til +2 mia).
    
- `long` → meget store heltal (fx op til 9 kvintillioner).
    
- `float` → decimaltal med _mindre præcision_ (fx 3.14, men kun ~7 cifre nøjagtighed).
    
- `double` → decimaltal med _høj præcision_ (fx 3.1415926535…). → det man oftest bruger til kommatal.
