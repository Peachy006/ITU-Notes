Når man bruger java math kan man gøre følgende ting:




```Java
Math.max(5, 10);
```
returnerer den højeste værdi



```Java
Math.min(5,10);
```
Returnerer den mindste af værdierne




```Java
Math.sqrt();
```
Kvadratroden af et tal





```Java
Math.abs(-2.4);
```

Returns the positive value, so a negative number will become positive and a positive number will not change


```Java
Math.random();
```

Giver et tilfældigt tal mellem 0 og 1, 1 ikke inkluderet.

Hvis du vil have et tal for eksempel mellem og 100:

```java
int randomNum = (int)(Math.random() * 101);  // 0 to 100
```
